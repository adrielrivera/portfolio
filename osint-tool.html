Setup Instructions

Prerequisites

  • Node.js (v14 or higher)
  • Python 3.8 or higher
  • Git

Backend Setup

Navigate to the backend directory:

cd backend

Create and activate a virtual environment:

python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate

Install dependencies:

pip install -r requirements.txt

Start the Flask server:

python app.py

The API will be available at http://localhost:5000

Frontend Setup

Navigate to the frontend directory:

cd frontend

Install dependencies:

npm install

Start the development server:

npm start

The application will open at http://localhost:3000

Core Features & Modules

1. Domain Intelligence (/domain)

  • I implemented WHOIS data analysis with fallback mechanisms
  • I developed DNS record enumeration (A, MX, TXT, etc.)
  • I created HTTP header analysis functionality
  • I provided mock data for major domains (Google, Facebook, Twitter)

2. Email Intelligence (/email)

  • I built email format validation with MX record checking
  • I simulated breach data (mimics HaveIBeenPwned API)
  • I developed domain-associated email discovery
  • I created professional reporting with confidence scoring

3. Username Research (/username)

  • I implemented social media presence detection across 30+ platforms
  • I created Sherlock-style username enumeration simulation
  • I designed deterministic results for consistent demonstrations
  • I developed professional confidence scoring and source attribution

4. IP Address Intelligence (/ip)

  • I integrated geolocation analysis using real APIs
  • I implemented WHOIS information lookup
  • I created reverse DNS resolution functionality
  • I simulated port scanning (mimics Shodan data)
  • I developed vulnerability assessment reporting
